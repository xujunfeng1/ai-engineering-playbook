# Contributing

Thanks for considering a contribution.

This project welcomes practical improvements to AI-assisted engineering workflows, especially templates, checklists, testing guidance, and small automation scripts.

## Contribution Guidelines

- Keep examples generic and free of private business data.
- Prefer simple Markdown templates over heavy process documents.
- Keep scripts portable and easy to run locally.
- Explain the engineering problem before proposing a new rule.
- Avoid vendor-specific assumptions unless the guidance is clearly optional.

## Pull Request Checklist

Before opening a pull request, please confirm:

- The change is reusable by more than one project.
- The documentation is clear and concise.
- Scripts were tested locally where applicable.
- No private paths, credentials, customer names, or internal URLs are included.
- The README or relevant docs are updated if behavior changed.
